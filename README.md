# Kaleidoscope-compiler
A compiler for the Kaleidoscope language. Written in C++ with lex/yacc and generate LLVM IR.

## Status
* Identifier: lower case alphanumeric string only.
* Literal:  numeric
* Operator: addition, subtraction, multiplication
